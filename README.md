# Homemade Cookies - Website

A simple, responsive website for homemade Malay and Singaporean cookies built with **HTML, CSS, and Tailwind CSS**.

## 🍪 Features

- 3 pages (Home, Products, Contact Us)
- Infinite scrolling reviews section
- Fully responsive design
- No JavaScript – just HTML & CSS

## 📁 Pages

- `index.html` – Home page with hero, features, and reviews
- `products.html` – Product listings
- `contact.html` – Contact form and info

## 🚀 Quick Start

Simply open `index.html` in your browser or use Live Server.

## 🎨 Tech Stack

- HTML5
- CSS3 (Tailwind CSS via CDN)
- Responsive design (Mobile, Tablet, Desktop)

## 🎬 Key Features

- **Blue gradient hero** – Eye-catching landing section
- **3-column feature cards** – Large Assortment, Fast Shipping, 24/7 Support
- **Infinite scroll reviews** – CSS-powered animation (no JS)
- **Mobile-friendly** – Works on all devices

## 🛠️ Customization

- Change colors in Tailwind classes
- Update text directly in HTML
- Adjust scroll speed in CSS `@keyframes`
- Swap images in `/images` folder

## 📝 Project Size

- Lightweight (no heavy dependencies)
- Fast loading
- Uses Tailwind CSS CDN
